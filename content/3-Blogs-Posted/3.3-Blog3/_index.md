# Summarizing the Journey of Building GreenLens Kids with First Cloud AI Journey

Hello everyone, we are IT students from HUTECH and VJIT, and we are currently participating in the First Cloud AI Journey program. Through the first two articles, we shared detailed experiences about optimizing the serverless architecture and the lessons we learned while building the application layer. In this third blog post, we would like to summarize the entire journey so far and the core values our team gained from the beginning until the project took shape.

## 1. Systems thinking determines product quality

Looking back at the whole project, we realized that even the most complete architecture diagram becomes meaningless without the ability to implement it. Security features such as AWS Cognito and Web Application Firewall only truly work when the team understands IAM permissions, manages environment variables securely, and controls resource usage carefully. The close connection between overall design thinking and detailed programming skills is the key that helped GreenLens Kids run smoothly and optimize costs.

## 2. Becoming sharper at handling incidents

Working on the project gave the team many valuable lessons through unexpected incidents. From handling the situation where cloud credit was exhausted and the cloud environment had to be reconfigured to a new account, to setting up Docker networking to run DynamoDB locally, and reviewing the causes of data inconsistency with the frontend team, these technical barriers forced the whole team to practice reading system logs, isolating problems quickly, and finding thorough solutions.

## 3. Growing through professional feedback

The project certainly would not have been completed without guidance from the advisory board. The mentors' strict, direct, and practical feedback helped the team completely change the way we approached technology. We learned to never compromise on security risks, and to always weigh cost carefully before deciding to integrate any new service into the system.

## Conclusion

The First Cloud AI Journey program has now come to an end, but the experiences we gained are truly valuable. The team had the opportunity to go through a complete product development process, from sketching the architecture and securing the system to handling infrastructure issues and integrating AI to complete the application. We would like to sincerely thank the organizers, the mentors, and all our teammates for working together to overcome countless technical challenges and bring GreenLens Kids to the finish line.
